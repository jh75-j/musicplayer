1.뮤직플레이어 개요
  - HTML,CSS,JAVASCRIPT를 활용하여 제작한 뮤직플레이어 애플리케이션입니다.
  - 사용자는 음악재생, 일시정지, 이전곡/다음곡 이동, 반복재생 설정, 볼륨조절등의 기능을 사용할 수 있으며 진행바를 통해 현재 재생위치 확인과 이동을 할 수 있습니다.

2. 사용기술
  HTML5 
  CSS3
  JAVASCRIPT
  Google Meterial Icons

3. 주요기능
   - 음악 재생 / 일시정지 
      play버튼 클릭시 음악 재생, pause버튼 토글설정   
   - 이전곡 / 다음곡 이동
      previous버튼은 이전 곡 이동, next버튼은 다음 곡 이동
   - 반복 재생 기능
      전체반복과 한복 반복 설정 토글 설정
   - 진행바 기능
       현재 재생시간 표시, 전체 재생시간 표기,
       진행바 클릭시 원하는 구간 이동 가능  
   - 플레이리스트 기능
      3곡이상 음악 등록
      리스트 클릭시 해당 곡 재생 
   - 앨범 및 곡 정보 변경
      곡 변경시 앨범 이미지 변경, 
      제목과 아티스트 정보 변경
   - 볼륨 조절 기능
      슬라이더를 이용한 볼륨 조절, 
      음소거 상태 아이콘 변경
